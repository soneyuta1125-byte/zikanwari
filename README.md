# じかんわり

子どものiPadスクリーンタイム管理PWAアプリです。

---

## ローカルサーバーの起動方法

ターミナルでこのフォルダに移動して、以下を実行してください：

```bash
npx serve .
```

起動後、表示されたURLをブラウザで開きます（例：`http://localhost:3000`）。

---

## iPadでの動作確認方法

1. Mac と iPad を同じ Wi-Fi に接続する
2. Mac のターミナルで `npx serve .` を実行する
3. 表示される「Network」のURL（例：`http://192.168.1.xx:3000`）をメモする
4. iPad の **Safari** でそのURLを開く

> **注意：** Chrome や Firefox では PWA 機能（ホーム画面追加・オフライン動作）が正しく動作しません。必ず Safari を使用してください。

---

## iPadのホーム画面への追加手順

1. iPad の Safari でアプリのURLを開く
2. 画面下部（または上部）の **共有ボタン（四角から上矢印のアイコン）** をタップする
3. メニューをスクロールして **「ホーム画面に追加」** をタップする
4. 名前を「じかんわり」のままにして **「追加」** をタップする
5. ホーム画面にアプリアイコンが追加される
6. 以降はホーム画面のアイコンから起動することで、フルスクリーンのアプリとして使用できます

---

## 設定パスコードの初期値

**1234**

設定画面から変更できます。4桁の数字で設定してください。

---

## 主な機能

- おにいちゃん・おとうとそれぞれのスクリーンタイム管理
- 平日・休日の自動判定（土日＝休日）
- 日付をまたいで自動リセット
- バックグラウンド中も壁時計ベースで正確に時間計測
- 残り5分・1分・時間切れで背景色が変化
- 親用設定画面（パスコード保護）
- オフライン動作対応（PWA）
